Cara Bermain

Space Explorer adalah game eksplorasi luar angkasa. Tujuan Anda adalah:

  • Jelajahi galaksi dengan menggerakkan kapal Anda menggunakan tombol navigasi
  • Pindai area untuk menemukan planet baru
  • Jelajahi planet untuk menemukan sumber daya dan barang langka
  • Selesaikan misi untuk mendapatkan pengalaman dan kredit
  • Tingkatkan kapal Anda untuk menjelajahi lebih jauh
  • Kumpulkan pencapaian dengan menyelesaikan tugas-tugas spesial

Gunakan kredit untuk membeli upgrade dan item. Kumpulkan XP untuk naik level dan membuka area baru!
